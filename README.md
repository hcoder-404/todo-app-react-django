# TaskFlow Pro

A modern full-stack To-Do application built with React and Django REST Framework. This app helps users manage daily tasks efficiently with a clean UI and powerful productivity features.

## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Search and filter tasks
- Due date management
- Dark/Light mode
- Responsive modern UI
- REST API integration
- Full-stack architecture using React + Django

## Tech Stack

### Frontend
- React
- JavaScript
- Axios
- CSS

### Backend
- Django
- Django REST Framework
- SQLite

## Installation

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## API Endpoint

```bash
http://127.0.0.1:8000/api/tasks/
```
